## Creating a New Model

1. Copy `Templates/Model Card.md` via Templater
2. Create folder: `My_Research/Models/{model-name}/`
3. Add `model-card.md` and `src/`
4. Link related papers with `[[@citekey]]`
5. Update [[00_Hub/Research Roadmap]]
